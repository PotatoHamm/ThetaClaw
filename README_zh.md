# ThetaClaw 🦞🛡️

*其他语言版本：[English](README.md)*

**ThetaClaw** 是一个基于 [OpenClaw](https://github.com/openclaw/openclaw) 构建的安全运行框架。本项目旨在保留原版个人 AI 助手强大能力的基础上，重构并引入全新的安全架构，为 AI 代理的执行提供更加坚固、可信赖的底层安全环境。

---

## 🙏 致谢与开源声明

本项目是对优秀开源项目 **OpenClaw** 的二次开发与安全加固版本（Fork）。
在此向 OpenClaw 的原作者及整个贡献者社区表示最诚挚的感谢！没有他们在 AI 助手领域的卓越基础工作，就不会有 ThetaClaw 的诞生。

* **原项目 GitHub 地址**：[openclaw/openclaw](https://github.com/openclaw/openclaw)
* **开源协议**：本项目继承并严格遵守原项目的 [MIT License](LICENSE)。
